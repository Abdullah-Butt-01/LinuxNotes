# Day 07 - Mini Project

## Objective

Practice core linux commands used in DevOps environments.

## Directory Structure
- configs/ - configuration files
- logs/ - logs directory
- scripts/ - automation scripts
- docs/ - documetation

## Tasks Performed
- Created project structure
- Edited configuration file using nano
- Secured file using chmod
- Simulated and killed a process
- Tested network connectivity using ping and curl
- Taken screenshots of terminal and moved them into docs/

## Key learnings
- Linux is file-based
- Permissions control accesss
- Process must be monitored
- Network and service checks are different 
- Images can be viewed using:
 - xdg-open (GUI systems)
 - copied to lecal systems if needed
