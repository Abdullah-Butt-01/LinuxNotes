# Day 02 - Linux Commands

**Date:** 12-Jan-2026

## Objective

Practice linux file operation commands

## Commands Practiced

* rm - delete files
  * rm -r - delete folder plus its contents
  * rm -i - ask before deletion

* cp source destination - copy files/folders
  * cp -r folder1 folder2

* cat filename - Quick view (Concatenate) - For short files

* less filename - For large files
  * q - quit

## what I did

* Created copy of files and folders, then deleted them
* Viewed files using cat and less
